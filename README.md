# crypto-regex

> regular expressions for cryptocurrencies

## regular expressions

 - bitcoin-address
 - bitcoin-uri
 - bitcoin-xpub-key
 - testnet-address
 - testnet-uri
 - testnet-tpub-key

## install project

`npm i crypto-regex --save`

## test project

`npm test`
